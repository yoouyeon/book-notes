## Derived Values

- 노트 파일 경로 : $ARGUMENTS[0]
- 현재 날짜 : !`date +"%Y-%m-%d"`

## Steps

1. 노트 파일을 읽는다.
2. 오타를 수정하고 사용자의 승인을 받아 변경사항을 적용한다.
3. 내용이 사실과 다른 부분이 있는지 확인한다.
  - 사실과 다른 부분이 있다면 올바른 사실과 함께 사용자에게 제안한다.
  - 내용 수정은 사용자가 직접 수정한다. (AI가 절대 직접 수정하지 않는다.)
4. 2번 단계에서 확인한 오타를 수정한 뒤 frontmatter의 `status` 필드를 변경한다.
  - 사실과 다른 부분이 있는 경우 : `reviewed`
  - 사실과 다른 부분이 없는 경우 : `completed`
5. status가 `completed`로 변경되고 frontmatter의 date 필드가 비어있는 경우, date 필드에 `$현재 날짜`를 기록한다. 

## Exceptions

- 노트 파일 경로가 입력되지 않은 경우 : "노트 파일 경로를 입력해주세요." 라는 메시지를 출력하고 종료한다.
- 노트 파일이 존재하지 않는 경우 : "노트 파일이 존재하지 않습니다." 라는 메시지를 출력하고 종료한다.

## Self-Check

- 노트 파일의 오타가 수정되었는가?
- 노트 파일의 사실과 다른 부분이 올바르게 수정되었는가?
- 노트 파일의 frontmatter의 `status` 필드가 올바르게 변경되었는가?
- 노트 파일의 frontmatter의 `date` 필드가 올바르게 기록되었는가?
