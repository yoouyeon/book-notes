## DML

**DML 명령어**

- `INSERT`
- `UPDATE`
- `DELETE`
- `MERGE`

## TCL

**`ROLLBACK`** : 트랜잭션에서 수행한 변경을 취소한다. `ROLLBACK TO 세이브포인트명`을 사용하면 해당 SAVEPOINT 지점까지의 변경만 취소한다.

## DDL

**DDL 명령어**

- `CREATE`
- `ALTER`
- `TRUNCATE`

**외래키 제약조건 추가** : 기존 테이블에 외래키 제약조건을 추가할 때는 `ALTER TABLE`을 사용한다.

```sql
ALTER TABLE 테이블명
  ADD CONSTRAINT 제약조건명 FOREIGN KEY(컬럼) REFERENCES 참조할테이블(컬럼);
```

**`ON DELETE CASCADE`** : 외래키에 지정하면, 부모 테이블의 행이 삭제될 때 자식 테이블의 관련 행도 연쇄적으로 삭제된다.
